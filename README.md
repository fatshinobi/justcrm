# JustCRM

## Ruby on Rails CRM application whith minimal (crazy minimal) functions.

The mission of JustCRM to be simple (as far as it's possible).

## System Requirements
  * Ruby 2.2
  * Ruby On Rails 4.2

## Important
  * JustCRM works for only single company. 
  * Only admin can add users.
